## Sample files for discussion

This directory holds samples for discussion, which may not be valid yet.
Because JSON data doesn't support comments, some explanation is given here.

## programme_surreytf_2016.json
This was derived from a spreadsheet used to create the programme at the Surrey Track and Field Championships.  It lists all events and their start times,
and which rounds they "feed forward" into.

There were no multi-events in this competition.
